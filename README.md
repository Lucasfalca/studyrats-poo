# StudyRats

Sistema de gamificação para estudos desenvolvido em Java utilizando Programação Orientada a Objetos.

## Objetivo

O projeto tem como objetivo incentivar estudantes a manterem uma rotina de estudos através de um sistema simples de pontuação e ranking.

Os usuários poderão registrar sessões de estudo, acumular pontos e acompanhar seu desempenho dentro da plataforma.

---

# Funcionalidades

- Cadastro de usuários
- Login de usuários
- Registro de sessões de estudo
- Sistema de pontuação
- Ranking de usuários
- Histórico de estudos
- Salvamento de dados em arquivos
